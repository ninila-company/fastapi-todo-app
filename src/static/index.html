        HELP: KEYBINDINGS

        Normal Mode

        • j / k - Навигация по задачам вверх/вниз
        • h / l - Навигация по колонкам влево/вправо
        • i - Войти в режим вставки (для создания задачи)
        • e или Enter - Редактировать выбранную задачу
        • v - Отметить задачу как выполненную/невыполненную
        • dd - Удалить выбранную задачу
        • :help + Enter - Показать этот экран

        Insert Mode

        • Enter - Создать новую задачу
        • h / l - Изменить срочность
        • Esc - Выйти в нормальный режим

        Edit Mode

        • j / k - Навигация по полям
        • h / l - Изменить срочность
        • Enter - Сохранить изменения
        • Esc - Отменить и закрыть окно
